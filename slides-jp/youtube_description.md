# YouTube動画概要欄

「原子は本当に存在するのか？」――120年前、これは真剣な科学論争でした

水に浮かべた花粉から飛び出た微粒子が不規則に動き続ける「ブラウン運動」。この謎の現象が、目に見えない原子の存在を証明する決定的な証拠になりました。

19世紀末、原子の存在は単なる「仮説」でした。直接見ることができないものを「実在」と認めるべきか？科学者たちは激しく議論していました。

この動画では、アインシュタインが1905年に発表した理論と、ペランによる精密な実験が、どのようにして「見えない原子」を「見える現象」にしたのかを解説します。
